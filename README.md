# nwiiter

느위터는 리덕스/리덕스 사가로 상태 관리 하였고 <br>
파이어베이스를 백엔드로 구현한 트위터 클론입니다. <br>

## Key Features

-   Sign Up / Sign In
-   Like / UnLike
-   Twitt / Del Twitt + Image Upload
-   Comment
-   User Profile
